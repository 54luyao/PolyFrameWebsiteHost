    Change Log

    PolyFrame 2

    What to expect in Version 2.3.0 (Release date TBD)

    • Create new force diagrams by merging individual force diagrams.
    • Matrix kinematic and static analysis for genetic structural forms (hybrid structural forms with bars and faces).
    • Linear optimization solver with different objective functions.

    Version 2.2.0 - 2024-02-18

    • A linear optimization solver as a supplementary tool for the algebraic workflow.
    • Bug fix.

    Version 2.1.5 - 2024-01-14

    • PFPipe Grasshopper component now bakes into Rhino with colors.
    • PFEdge, PFFace, PFVertex, PFCell parameters now has auto-cast feature into geometry types.

    Version 2.1.4 - 2023-11-26

    • Added supports to Rhino 8.

    Version 2.1.3 - 2023-05-29

    • Bug fix.

    Version 2.1.2 - 2023-05-05

    • Bug fix.

    Version 2.1.1 - 2023-05-04

    • Bug fix.

    Version 2.1.0 - 2023-05-03

    • Added Example Files component with embedded gh scripts.

    Version 2.0.0 - 2023-04-28

    • Initial release of PolyFrame 2, supports Rhino 6 and Rhino 7.
    • Includes all legacy Rhino commands, plus a lot more powerful functionalities delivered through grasshopper.
    • A unified MSI installer that installs plugin for both Rhino and Grasshopper.
    • Automatically detects the Rhino versions installed and create integration accordingly.
    • The new algebraic workflow greatly increases speed, precision, and tension-compression form-finding capability.
    • Data structure made accessible through Grasshopper components.
    • Data structure made accessible through Python and C# scripting interfaces.
    • Matrix analysis for kinematic and static indeterminacy of polyhedral structures.
    • Minkowski transformation integration between form and force diagrams.
    • Force diagram subdivision built-in patterns.
    • Smooth data transition between Rhino and Grasshopper.
    • Shellular-funicular form-finding capability.
    • And a lot more...
