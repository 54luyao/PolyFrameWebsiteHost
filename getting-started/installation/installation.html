    Installation

    Ovenbird is a plug-in for Grasshopper, the graphical algorithm editor for the 3D modeling software of Rhino. Ovenbird works on Grasshopper for Rhino ≥ 7 for both Windows and Mac. (1)

    1. Currently Overhang Optimization is limited to Windows.

    Download Ovenbird from food4Rhino. To install, unzip the package. In Grasshopper, go to File → Special Folders → Components Folder. Save the unzipped folder there and delete the previous one if you have it. Make sure the .gha file is unlocked. Restart Rhino and Grasshopper.
